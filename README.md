## Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/Akash-Jambulkar/RBAC-Note-System
cd rbac-notes-system
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. **Default Admin Credentials**
```
Username: admin
Password: admin123
```
